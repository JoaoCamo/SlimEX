SlimEX

Os jogos também possuem uma explicação de como jogar dentro de suas páginas

Slime Adventure

O objetivo é pular os obstaculos que estão se aproximando

Para escolher a dificuldade utilize a seta da esquerda e direita

A tecla W faz com que a slime pule

Slime Eater

O objetivo é comer o ponto azul

Para navegar pelo menu utilize as setas do teclado

Para mover a slime:

W faz a slime ir para cima

A faz a slime ir para esquerda

S faz a slime ir para baixo

D faz a slime ir para direita
